# AI Green IDC 대시보드 — UI/UX 설계 명세서

## 1. 개요

| 항목            | 내용                        |
| --------------- | --------------------------- |
| 프레임워크      | Streamlit 1.55.0            |
| 차트 라이브러리 | Plotly (go / make_subplots) |
| 진입점          | `apps/dashboard/app.py`     |
| 레이아웃        | Wide mode (`layout="wide"`) |
| 사이드바        | Expanded (초기값)           |

## 2. 전체 페이지 구조

```
app.py  (st.navigation)
├── 0_대시보드.py    dashboard  대시보드   (구 홈 + 실시간현황 통합)
├── 1_운영_관리.py   tune       운영 관리  (구 냉각제어 + 위기시나리오 통합)
├── 2_ESG_지표.py    eco        ESG 지표
└── 3_분석_도구.py   analytics  분석 도구  (구 예측애니메이션 + 존히트맵 탭 통합)
```

> 페이지 아이콘: Material Symbols (`:material/icon_name:`) 사용

## 3. 공통 사이드바 (`sidebar.py`)

모든 페이지는 `render_sidebar()`를 호출하며 동일한 사이드바를 공유한다.

### 3-1. 섹션 구성

```
⚙️ 시뮬레이션 파라미터
  ├── [selectbox] 시나리오 — 여름 / 봄·가을 / 겨울
  │
  ├── 서버 구성
  │   ├── [slider] CPU 서버 대수 (100~1000, 기본 400, step 50)
  │   ├── [slider] GPU 서버 대수 (0~200, 기본 20, step 10)
  │   └── [slider] 평균 CPU 사용률 (10~100%, 기본 60%)
  │
  ├── 냉각 설정
  │   └── [slider] CRAH 공급 온도 (14~24°C, 기본 18°C)
  │
  ├── 🚨 위기 시나리오
  │   └── [button × 4] 정상 | 서버급증 | 냉각고장 | 폭염
  │       → 위기 모드 활성 시 🔴 에러 배너 / 비활성 시 🟢 성공 배너
  │
  └── 🔌 서비스 상태
      └── 마이크로서비스별 🟢/🔴 상태 표시 (10초 TTL 캐시)
```

### 3-2. 반환 데이터 (`dict`)

사이드바는 시뮬레이션 실행 후 아래 키를 포함한 dict를 반환한다.

| 키                                           | 타입              | 설명                              |
| -------------------------------------------- | ----------------- | --------------------------------- |
| `scenario`                                   | str               | 선택된 시나리오명                 |
| `num_cpu` / `num_gpu`                        | int               | 서버 대수                         |
| `cpu_util`                                   | float             | CPU 사용률 (0.0~1.0)              |
| `supply_temp`                                | int               | 공급 온도 (°C)                    |
| `crisis_mode`                                | str \| None       | 위기 모드 키                      |
| `cfg`                                        | dict              | 위기 모드 설정 (`CRISIS_CONFIGS`) |
| `df`                                         | DataFrame         | 시뮬레이션 결과 (24행)            |
| `df_full`                                    | DataFrame         | 전체 24h 데이터 (애니메이션용)    |
| `df_base`                                    | DataFrame \| None | 위기 모드 비교용 정상 데이터      |
| `avg_pue` / `avg_cop`                        | float             | 24h 평균 PUE / COP                |
| `avg_it` / `avg_cooling` / `avg_total`       | float             | 전력 평균 (kW)                    |
| `peak_return_temp` / `max_return_temp`       | float             | 환기 온도 (°C)                    |
| `current_mode`                               | str               | 피크 시점 냉각 모드               |
| `current_outdoor`                            | float             | 피크 시점 외기 온도               |
| `peak_it_power`                              | float             | 피크 IT 전력 (kW)                 |
| `rack_labels` / `rack_temps` / `rack_colors` | list              | 랙 온도 시각화 데이터             |
| `over_threshold`                             | int               | 경고 기준 초과 랙 수              |
| `esg` / `esg_base`                           | dict              | ESG 지표 (탄소·비용·물)           |
| `avg_pue_base`                               | float \| None     | 정상 모드 기준 PUE                |
| `ctrl_rule` / `ctrl_rl`                      | dict              | Rule-Based / RL 제어 추천값       |

### 4-6. 🔮 예측 애니메이션 (`5_예측_애니메이션.py`)

**목적**: 24h 예측 결과 타임랩스 재생. (목업 — Forecast Service 연동 예정)

```
[사이드바 추가 섹션]
  🎬 재생 컨트롤
    [select_slider] 재생 속도 — 매우빠르게(0.1s) / 빠르게(0.2s) / 보통(0.4s) / 느리게(0.8s)
    [▶ 재생] [⏸ 정지] [⟲ 초기화] — 3열 버튼

[🚧 목업 배너]

[헤더] — 5:1 컬럼
  좌: 현재 시각 표시 (HH:00, X/24 프레임) 또는 안내 캡션
  우: ▶ 재생 / ⏸ 정지 토글 버튼 (primary)

[진행 바] — st.progress (0~1)

[KPI 카드] — 5열 (애니메이션 활성 시)
  PUE | IT 전력 | 칠러 전력 | 냉각 모드 | 외기 온도

[2열 차트]
  ├── 전력 추이 (현재 프레임까지 + 전체 24h 흐린 배경)
  └── PUE 추이 (현재 프레임까지 + 전체 24h 흐린 배경)
      → 현재 시각 빨간 수직선 표시

[📋 누적 데이터 expander] — DataFrame
```

**프레임 전진 로직**

```
anim_running == True
  → time.sleep(anim_speed)
  → anim_hour < 23: anim_hour += 1
  → anim_hour == 23: anim_running = False + 완료 알람
  → st.rerun()
```
